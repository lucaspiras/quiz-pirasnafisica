Configurações

Cada pergunta pode ter um tempo próprio; este é o valor usado quando você não define um.

Com o bônus ligado, quem acerta ganha os pontos da pergunta + um extra proporcional ao tempo que sobrou. Vale no jogo ao vivo e no modo praticar.

Exemplo: pergunta de 100 pontos com bônus de 50% — acerto imediato vale 150; acerto no último segundo vale 100.

Banco de perguntas público

Busque perguntas já aprovadas de outros quizzes, marque as que quiser e importe todas de uma vez. Perguntas que você já importou não aparecem de novo.

Importar arquivo

Envie um arquivo .json com uma lista de perguntas para adicionar todas de uma vez.

Formato: um array de objetos { text, options: [{key,label}], correct_key, points, media_url, media_type, question_seconds, tags }. Só text, options e correct_key são obrigatórios.

Hospedar ao vivo

Ao hospedar, você recebe um código de 6 dígitos para compartilhar com os participantes.

Como funciona: você cai no painel de controle (link secreto, não compartilhe) e de lá abre o telão para projetar. Os participantes entram pelo celular com o código ou o QR code — sem instalar nada.
